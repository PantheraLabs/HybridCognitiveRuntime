# Change Log

All notable changes to the "HCR Resume" extension will be documented in this file.

## [0.1.0] - 2026-04-30

### Added
- Initial release with core session resume functionality
- Automatic context restoration on window focus
- File change tracking with automatic state updates
- Git state integration (commits, branches, uncommitted changes)
- AI-powered task inference and progress tracking
- Next action recommendations with confidence scoring
- Configurable auto-resume with idle threshold
- Status bar showing current task and progress
- "HCR: Resume Session" command palette integration
- Heartbeat mechanism for keeping engine connection alive
- Active tab tracking for precise file context

### Features
- **Smart Session Resume**: Restore full cognitive context in <10 seconds
- **Token Efficiency**: 10-100x reduction vs traditional AI context rebuilding
- **Causal Analysis**: Predict ripple effects of code changes
- **Cross-Session Memory**: Persistent state across IDE sessions
